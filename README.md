寻味 - 城市美食探店推荐网站
项目简介
寻味是一个专注于城市美食探店推荐的平台，帮助本地人和外来游客快速找到靠谱、不踩雷的好吃餐馆。网站采用简约清新的暖色调设计风格，提供美食店铺推荐、探店日记分享、智能筛选搜索、美食排行榜、收藏与打卡等核心功能。

技术栈
前端技术: HTML5 + CSS3 + JavaScript (原生)
设计风格: 简约清新、暖色调、响应式布局
图标: Emoji + 在线图片资源

页面说明
1. 首页 (index.html)
暖色调首页设计，美食大图封面
导航栏、推荐店铺、热门榜单入口
分类导航（火锅、烧烤、甜品、中餐、西餐等）
探店日记预览
2. 店铺列表页 (shop.html)
智能筛选搜索（菜系、商圈、人均、特色）
排序功能（综合、评分、价格、距离）
店铺卡片列表展示
分页功能
3. 店铺详情页 (shop-detail.html)
店铺图片画廊
店铺信息（地址、电话、营业时间）
推荐菜品展示
用户评价列表
收藏/打卡按钮
4. 探店日记列表页 (diary.html)
日记列表展示
筛选标签（推荐、最新、热门、关注）
热门话题推荐
推荐用户关注
5. 发布探店日记页 (diary-publish.html)
店铺信息填写
评分系统（总体、口味、环境、服务、性价比）
图片上传功能
标签添加
6. 美食排行榜页 (rank.html)
热门榜、新店榜、性价比榜、口味榜、环境榜
TOP 3 突出展示
榜单切换功能
7. 收藏/打卡页 (collect.html)
个人中心头部信息
想去/已打卡/收藏列表
探店统计图表
最近浏览记录
我的日记
8. 登录/注册页 (login.html)
登录/注册切换
表单验证
第三方登录入口（微信、微博）
9. 管理员后台页 (admin.html)
数据概览仪表盘
店铺管理（增删改查）
日记审核（通过/拒绝）
用户管理
评论管理
系统设置
核心功能
美食店铺推荐: 按菜系、商圈、人均价格、评分分类展示
探店日记分享: 发布探店文案、打卡照片、真实用餐评价
智能筛选搜索: 多维度精准查找
美食排行榜: 实时更新热门榜单
收藏与打卡: 标记"想去、已打卡"，方便选店
管理员后台: 审核帖子、管理店铺信息、屏蔽恶意差评
设计特点
简约清新: 暖色调为主，排版干净舒服
响应式设计: 电脑端、手机端自适应浏览
无弹窗广告: 纯净的用户体验
精致杂志风: 像美食杂志一样的视觉体验
使用说明
直接在浏览器中打开 index.html 即可访问网站
所有页面均为静态HTML，无需后端服务器
数据使用模拟数据展示，实际应用中可接入后端API
